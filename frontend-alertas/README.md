# Frontend Alertas Médicas

Frontend Angular con login mediante Azure AD B2C/MSAL y consumo del BFF Spring Boot.

## Ejecutar local

```bash
npm install
npm start
```

URL local: `http://localhost:4200`

## Funcionalidades

- Login y logout con Azure AD B2C.
- Dashboard protegido.
- Registro de pacientes con validaciones.
- Listado y eliminación de pacientes.
- Registro de signos vitales por paciente.
- Visualización de alertas activas.
- Marcar alertas como atendidas.
